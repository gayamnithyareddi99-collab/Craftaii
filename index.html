How Craftaii Works

Craftaii follows a simple workflow to detect visible construction defects from site images and generate basic inspection evidence.

01

Upload Site Images

Site images are uploaded from mobile devices or cameras during inspections.

02

AI Detects Defects

The system analyzes images to identify visible workmanship issues such as missing elements or poor construction quality.

03

Review Results

Users review detected issues and validate results before finalizing inspection outcomes.

04

Generate Basic Report

A simple report is generated with detected defects to support inspection records and basic compliance documentation.

Frequently Asked Questions

Learn how Craftaii is being developed to detect visible construction defects from site images and support inspection workflows.

What is Craftaii?

Craftaii is an MVP-stage platform focused on detecting visible construction defects from site images using AI.

How does Craftaii work?

Users upload site images, and the system analyzes them to identify visible defects. The results can then be reviewed and used to generate basic inspection reports.

Who can use Craftaii?

Construction teams, engineers, and small contractors can use the platform to support inspections and identify issues early.

What stage is the product in?

Craftaii is currently in the MVP stage, focusing on testing image-based defect detection and improving the system through pilot projects.

Does it support full automation?

No. The current system supports basic detection and requires manual review to validate results before reporting.

How can I start using it?

You can request a pilot to test the MVP and provide feedback for further development.
